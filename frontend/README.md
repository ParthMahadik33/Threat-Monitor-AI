# Threat Monitor AI - Frontend

Modern React + TypeScript frontend for the Threat Monitor AI security monitoring platform.

## Setup

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run test` - Run tests
- `npm run lint` - Run ESLint
